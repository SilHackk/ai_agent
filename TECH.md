- Python

- PyMuPDF (PDF parsing)

## Architecture

User Input:
- Email text
- PDF file

↓

Processing:
- Extract PDF text

↓

AI Agent:
- Classification
- Information extraction
- Missing data detection
- Response generation

↓

Output:
- Structured UI display
