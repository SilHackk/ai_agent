# Design

## UI Goals
- Simple and clean interface
- Fast workflow for employees
- Clear structure of results

## Layout

### Input section
- Email text box
- PDF upload

### Action
- "Analyze request" button

### Output sections
Displayed as structured blocks:

- Category
- Extracted data
- Missing information
- Email response draft
- Internal summary

## UX principles
- No clutter
- Clear hierarchy
- Easy copy/paste for email
- Focus on speed

## Future improvements
- Copy buttons for each section
- Editable response before sending
- Highlight important info from PDF
